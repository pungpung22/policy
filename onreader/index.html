What OnReader Does

  • Imports user-selected local files into an on-device library.
  • Supports reading and viewing common text, ebook, PDF, image, archive, audio, video, and office document formats where iOS allows it.
  • Stores bookmarks, reading position, notes, reader settings, TTS settings, and library organization on the device.
  • Offers optional AI reading help only when the user starts an AI action.
